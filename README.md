## üìö Docker Concepts Covered

### 1. Docker Installation and Setup
- Install Docker Desktop  
- Create and use Docker Hub account  

### 2. Basic Docker Commands
- `pull`, `run`, `exec`, `stop`, `start`, `rm`, `rmi`  

### 3. Docker Image Management
- Build and push custom Docker images  

### 4. Dockerfile Instructions
- `LABEL`, `ADD`, `COPY`, `ARG`, `RUN`, `EXPOSE`, `ENV`, `CMD`, `WORKDIR`, `ENTRYPOINT`, `HEALTHCHECK`, `USER`  

### 5. Docker Ports
- Mapping ports using `-p` and `-P`  

### 6. Docker Volumes and Data Management
- Basics, usage with containers, populating data, and sub-directory mounts  
- Difference between **Volumes** and **Bind Mounts**  

### 7. Docker Bind Mounts
- Basics, Read-Only Mode, Non-Empty Directory Use Cases  

### 8. Temporary File Storage
- Using `tmpfs` mounts  

### 9. Multi-container Applications
- Deploy multi-container setups using Docker commands  

### 10. Docker Compose
- Basics, Named Volumes, Multi-container apps, Scaling, Networks, Health Checks, Profiles, and more  

### 11. Advanced Compose Features
- Network configurations, aliases, links, build, and develop-watch modes  

### 12. Docker BuildKit
- Buildx CLI, Local and Cloud Builders, Multi-Platform Builds  

### 13. Docker Cloud Build
- Remote builds using Docker Build Cloud  

### 14. Docker Multi-stage Builds
- Optimize and reduce image size using multi-stage build strategies
